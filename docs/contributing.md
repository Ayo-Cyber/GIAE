# Contributing

The canonical contributor guide lives in the repository root:
**[CONTRIBUTING.md](https://github.com/Ayo-Cyber/GIAE/blob/main/CONTRIBUTING.md)**.

That document covers:

- Development setup
- Branching and commit conventions
- Code style (ruff + mypy + pytest)
- Test requirements (unit, integration, validation suite)
- Pull request process
- How to add a plugin
- How to report bugs and propose features

It also links to the [Code of Conduct](https://github.com/Ayo-Cyber/GIAE/blob/main/CODE_OF_CONDUCT.md)
and [Security Policy](https://github.com/Ayo-Cyber/GIAE/blob/main/SECURITY.md).

If you're looking for *how to extend GIAE* (plugins, ORF finders,
new evidence types), see **[Extending GIAE](extending.md)** instead.
